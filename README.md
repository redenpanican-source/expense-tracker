# Expense Tracker

A personal expense logging app built with React and Vite.

## Features

- Log expenses by date, category, amount, and description
- Month-by-month navigation with a running total
- Category breakdown with visual bars
- Export expenses to CSV
- Dark green and gold UI theme

## Tech Stack

- React 19 + Vite
- Inline styles (no CSS framework)
- Google Fonts: Fraunces, Sora, Spline Sans Mono

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed via [Vercel](https://vercel.com). Push to `main` to trigger a deploy.
